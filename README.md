# Setup Guide
- Clone the project.
- Run following commands in the terminal:
- Create `.env` file in root folder, WIth properties defined in `.env-example`.
```bash

npm start
``` 
- Then use url to get data `http://{HOST}:{PORT}/api/employees/{id}`
